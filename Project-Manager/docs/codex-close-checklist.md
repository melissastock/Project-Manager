# Codex Close Checklist (Fast Pass)

Use this at the end of every session. Timebox: ~2–3 minutes.

---

## 1. Evidence Collected
- [ ] Commits listed (with SHA)
- [ ] Files changed listed
- [ ] Commands/scripts noted (if relevant)
- [ ] Tests/validation noted (if any)

---

## 2. PM-Ready Summary
- [ ] What changed (facts only)
- [ ] Current state
- [ ] Blockers/risks
- [ ] Top 1–3 next actions

---

## 3. Fact Discipline
- [ ] Facts are verifiable
- [ ] Interpretations labeled
- [ ] Opinions labeled

---

## 4. Review Gates
- [ ] Code review complete
- [ ] QC / validation complete
- [ ] Governance / legal / privacy review complete
- [ ] Blockers fixed or explicitly accepted

---

## 5. Context for Next Dev
- [ ] Where to look first
- [ ] Key decisions captured
- [ ] What not to change (if applicable)

---

## 6. System Impact
- [ ] PM updates needed identified:
  - [ ] repos.json
  - [ ] STATUS.md
  - [ ] dependency docs
  - [ ] none

---

## 7. File Created
- [ ] Handoff file saved to:
  `docs/session-handoffs/YYYY-MM-DD-[project]-handoff.md`

---

## 8. Final Check
- [ ] Another developer could pick this up without guessing
- [ ] Nothing critical left undocumented

---

## Done
If all boxes are checked → session can be closed

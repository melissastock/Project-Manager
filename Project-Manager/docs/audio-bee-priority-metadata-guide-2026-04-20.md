# Audio Bee Priority Metadata Guide - 2026-04-20

Use this guide with `docs/audio-bee-priority-metadata-intake-2026-04-20.csv`.

## Goal

Finalize metadata for priority anchor files so they become searchable and publication-ready in a controlled way.

## Field Rules

- `review_status`: `todo`, `in_review`, `done`
- `canonical_title`: final human-facing title for search and publishing
- `rights_status`: `cleared`, `restricted`, `unknown`
- `publishable`: `yes`, `no`, `needs_edit`
- `sensitivity_level`: `low`, `moderate`, `high`
- `primary_topics`: short comma-separated keywords
- `people_entities`: people/orgs explicitly present in content
- `summary_for_search`: 1-2 sentence factual description

## Suggested Order

1. Complete `rights_status` and `publishable` first.
2. Then set `canonical_title` and `summary_for_search`.
3. Add topics/entities after transcript review.

# Project Manager

This repository is the top-level portfolio manager for the projects inside this workspace.

It is intended to track the overall project map while preserving the individual child repositories as independent git repositories. Nested repositories will appear here as gitlinks when staged from the top level.

## Purpose

- Maintain a single workspace-level view of the portfolio
- Track top-level planning and organization files
- Preserve each child project's own history and branch structure
